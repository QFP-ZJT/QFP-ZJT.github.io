计算机组成原理课程设计

设备参数

  1. ROM:25683
  2. RAM:256*8

设计方案

硬件组成部分

CPU内部:
  1. 节拍发生器(分为大节拍和小节拍 暂定4个小节拍刺激一个大节拍)
  2. 寄存器组(包括固定的PC,IR,MAR,MDR寄存器)
  3. CU(指令译码与微程序实现)
  4. ALU模块
  5. 总线
CPU外部
  1. ROM(存储微程序)
  2. RAM(指令与数据)

运行步骤

  1. 清空PC寄存器
  2. 大节拍发生器生成节拍，刺激PC增1
  3. CU译码由小节拍刺激，完成该条命令的需要完成的任务
  4. 大节拍增1，继续运行

设计部分

指令可以实现的功能
  • 取指
  • 从RAM中读
  • 向RAM中写
  • 两个数相加
  • 两个数相减
  • 两个数相乘
  • 寄存器间数据转移
  • 输出到指示灯
微程序指令实现部分 (由小节拍控制从ROM中读取指令控制执行,RAM默认读有效)
取值
  • T0:PC -> MAR
  • T1:M(MAR) -> MDR $\overline{RD}$
  • T2:PC+1 -> PC
  • T3:MDR -> IR
从RAM中读
  1. 将MAR或者PC中的地址输出到RAM中,将读信号线置为有效
  2. MDR锁存数据
  3. 将MDR中的数据转移到指定寄存器
向RAM中写
  1. 将数据转移到MDR中
  2. 将MAR中的地址输出到RAM中，
