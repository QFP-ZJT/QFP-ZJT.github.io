如何在PhpStorm中关联gitHub项目

一、从GitHub上下载项目

  • 在PhpStorm中找到/VCS/checkout from version control/Github选项
  • 在弹出的文本框中的Git Repository URL中填入https://github.com/QFP-ZJT/qlsoftware.git第二行选择存放于本地的哪个目录下，第三行为项目的名称，我们统一称为qlsoftware就可以了
  • 点击clone即可将项目下载到本地

项目已经初始化，请以Github上的作为标准

二、提交你的更改

  • 在PhpStorm中找到/VCS/Git/commit file
  • 标号1为你所做出的更改的内容

标号2为填写你所更改的原因大家一定要填写，方便管理需要备注上自己的姓名，否则查出来杀无赦
